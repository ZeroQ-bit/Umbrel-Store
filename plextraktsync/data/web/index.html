PlexTraktSync for Umbrel

Import Trakt Continue Watching into Plex.

This app runs PlexTraktSync with Trakt-to-Plex playback progress enabled by default. Ratings, watchlists, collections, liked lists, and watched-status syncing are disabled until you edit the config.

Runs one sync on app start after login

1. Connect Plex and Trakt

  1. SSH into your Umbrel.
  2. Run the login command below.
  3. Follow the Plex and Trakt prompts.
~/umbrel/scripts/app compose plextraktsync exec plextraktsync plextraktsync login

If Plex is on this Umbrel, use http://host.docker.internal:32400 when asked for the Plex server URL.

2. Run the import

Restart the app after login. PlexTraktSync will run sync once and then stay idle.

~/umbrel/scripts/app restart plextraktsync

To sync again later, restart the app or run:

~/umbrel/scripts/app compose plextraktsync exec plextraktsync plextraktsync sync

Config location

The persistent PlexTraktSync config is stored inside this Umbrel app's data folder.

plextraktsync/data/config/config.yml

The seeded config enables only:

sync:
  trakt_to_plex:
    playback_status: true

Upstream project

PlexTraktSync is maintained at github.com/Taxel/PlexTraktSync.

If you want full two-way syncing later, edit config.yml carefully and restart the app.
